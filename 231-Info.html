Permutations of the Tree:  BOOK 231


This is a data CD containing the complete BOOK 231.  You can find the main body of BOOK 231 on my website [click here], but it was too extensive a work to post the whole of it.  It would have taken an entire website all of its own and have been viewable only with a DSL/Broadband connection!  ;-)  The printed, paper version (completed in early March of 1999) is approximately 1,000 pages long and since it is such an obscure work, I decided that the only way for me to present the whole work to the public was in a data CD format.

The presentation is set up like a website, done in .html format, and includes numerous thumbnail galleries and slide shows.  Works best with computers running a Windows system - some Mac users have reported difficulties with the slide shows.

This is a very complex kabbalistic work, designed with the advanced kabbalist in mind.  Before ordering this disk, I strongly suggest that you take a look at the presentation available on my website and decide if it's something you're interested in.
